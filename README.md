# Mobile Banking Quality Assurance Project

Repositori ini berisi dokumentasi lengkap QA (Quality Assurance) dan proyek otomatisasi pengujian untuk aplikasi Mobile Banking (Livin' by Mandiri). 
Proyek ini mencakup seluruh siklus pengujian, mulai dari perencanaan, pembuatan test case, pelaporan bug, hingga skrip otomatisasi menggunakan Katalon Studio.

- **Manual Testing**: Kumpulan test case mendetail untuk fitur kritis seperti Login, Transfer Rekening, Bayar, dan Top Up.
- **Bug Reporting**: Laporan bug terstruktur yang mencakup temuan pada fitur transfer, pembayaran, dan QR.
- **Automation Testing**: Skrip otomatisasi siap pakai untuk alur Login dan Transfer menggunakan Katalon Studio.
- **Test Summary**: Laporan akhir yang merangkum hasil eksekusi pengujian dan kualitas aplikasi secara keseluruhan.
